# Tenda Vulnerability

Vendor: Tenda

Product: AC18

Version: V15.03.05.19_multi

Type: Stack Overflow

Author: yanzhi Chen

Institution: chenyanzhi24@mails.ucas.ac.cn



## Vulnerability description

We found an stack overflow vulnerability in Tenda router with firmware which was released recently, allows remote attackers to crash the server.

**Stack Overflow**

In `httpd` binary:

In `formSetSysToolDDNS` function, `ddnsPwd` is directly passed by the attacker,  and its value is written to `adv.ddns1.pwd`.

![image-20260614201307595](./images/1/image-20260614201307595.png)

In `formGetSysToolDDNS` function, The value of `adv.ddns1.pwd` is written to `v14`, which is only 32 bytes long. If this part of the data is too long, it will cause the stack overflow, so we can control the `ddnsPwd` to crash the server.

![image-20260614201533108](./images/1/image-20260614201533108.png)



**Supplement**

In order to avoid such problems, we believe that the string content should be checked in the input extraction part.
